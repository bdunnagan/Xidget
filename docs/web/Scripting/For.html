for

Overview

The if script element works as either a numeric iterator, or a node-set iterator. When the source attribute or element is present, it contains an xpath that returns the node-set to be iterated. The assign attribute contains the name of the variable to which each element of the node-set will be assigned.

Numeric iteration requires the from and to attributes, and optionally the by attribute. The from attribute contains a numeric xpath that gives the starting value. The to attribute contains a numeric xpath that gives the ending attribute (inclusive), and the by attribute contains a numeric attribute that gives the increment.

Iterating A Node-Set

The following example prints each node in the node-set given by the source xpath:

<for assign="node" source="collection('library')/books/book"> <print>$node</print> </for>

Numeric Iteration

The following example prints the numbers from 1 to 10:

<for assign="i" from="1" to="10"> <print>$i</print> </for>

The following example prints the odd numbers from 1 to 10:

<for assign="i" from="1" to="10" by="2"> <print>$i</print> </for>
