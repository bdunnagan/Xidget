If/Else

Overview

The if script element conditional executes a script. If the if condition is not met then the next else script element will be executed. An xpath defines the condition under which the script will be executed. The xpath is evaluated as if it were enclosed by the boolean() function, so any xpath return type is valid.

There are three different attributes in which the xpath condition may be specified. If the condition is specified in the test or true attributes then the script will be executed if the boolean value of the xpath is true. If the condition is specified in the false attribute then the script will be executed if the boolean value of the xpath is false.

Keep in mind that almost all script elements may be made conditional by defining the when attribute. The when attribute contains an xpath which is evaluated as a boolean. Therefore, it is only necessary to use the if script element when you need an else script, too.

Using An If Statement

The following example will print the string "Frank Wilzcek":

<assign name="book">collection('library')/books/book[ title='The Lightness of Being']</assign>
<if test="$book/title = 'The Lightness of Being'">
   <print>$book/author</print>
</if>

The following example will also  print the string "Frank Wilzcek":

<if true="$book/title = 'The Lightness of Being'">
   <print>$book/author</print>
</if>

These examples are equivalent to the following:

<print when="$book/title = 'The Lightness of Being'">$book/author</print>

Using If/Else Statements

The following example will print the string "John Irving":

<assign name="book">collection('library')/books/book[ title='The Cider House Rules']</assign>
<if test="$book/title = 'The Lightness of Being'">
   <print>$book/author</print>
</if>
<else>  
   <print>$book/author</print>
</else>

Using Negation

The following example will not print anything:

<assign name="book">collection('library')/books/book[ title='The Lightness of Being']</assign>
<if false="$book/title = 'The Lightness of Being'">
   <print>$book/author</print>
</if>
