Xidget Reference

Element Description
Window A top-level window xidget.
Form A general purpose xidget container.
Tabs A tab container with one tab for each form child.
Label A text and/or image label xidget.
Button A push button, check button, toggle button or radio button.
Combobox A combo-box or choice list xidget.
Slider A numerical value xidget.
Text A single or multi-line text xidget.
Table A table xidget.
Tree A tree xidget.
Menu A top-level menu xidget used in menubars, toolbars and popup menus.
MenuItem An item in a menu.
