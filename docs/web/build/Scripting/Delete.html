delete

Overview

The delete script element removes one or more elements from their respective parents. It is important to note that the deleted elements may still be referenced by variables. The element is not necessarily "gone forever". It has only been removed from the model in which it resided.

In addition, the element may still be bound to one or more xpaths if the context in which the xpath was bound resided within the sub-tree of the element. If, and only if, the element is reachable from the context in which the xpath was bound then the element will still be bound to the xpath.

Removing One Element From Its Parent

The following example removes one element from its parent.

<delete>collection('library')/books/book[ title='The Lightness of Being']</delete>

Removing Several Elements From Their Respective Parents

The following example removes several elements from their respective parents.

<delete>collection('library')/books/book</delete>
