select

Overview

The select script element executes one of several scripts depending on the string result of the xpath contained in the source attribute or element. The select element contains one or more case elements and an optional default element. Each case element has an id attribute which determines the case that is selected. The string result of the source attribute or element is mapped to a case script by its id.

Basic Mapping

The following example will print "John Irving":

<assign name="book">collection('library')/books/book[ title='The Cider House Rules']</assign> <select source="$book/author"> <case id="John Irving"> <print>'John Irving'</print> </case> <case id="Frank Wilzcek"> <print>'Frank Wilzcek'</print> </case> </select>
