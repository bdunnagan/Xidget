OnKey

Overview

The onKey script can be added to any xidget. It is called when a specified key sequence is pressed. The key sequence is defined by an xpath expression that returns a string. The string consists of zero or more modifier keys followed by one or more non-modifier keys. For example, the following key sequence will activate the script when the "control" and "c" keys are pressed.

<onKey keys="'control,c'"> <print>'CONTROL C'</print> </onKey>

Each modifier key is separated by a "+". Non-modifier keys are separated from the last modifier and from each other by a ",". The following list gives some examples:

shift+control,c control,a,b a f1 enter
