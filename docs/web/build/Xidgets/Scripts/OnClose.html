OnClose

Overview

The onClose script can be added to any xidget. It is called after the xidget is destroyed.

See The Xidget Lifecycle for more information.

In the following example, the onClose script saves edited data to a file. Each time the form is closed it will replace the content of the file with whatever was typed into the text field.

<form> <onOpen> <create assign="content" name="'content'"/> </onOpen> <onClose> <fileSave source="$content" overwrite="true">'myFile.xml'</fileSave> </onClose> <layout> <layoutFillX/> <layoutFillY/> </layout> <text> <label>'Content'</label> <source>$content</source> </text> </form>
