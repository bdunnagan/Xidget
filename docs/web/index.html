What is a Xidget?

A xidget is an fragment of XML that describes a graphical user-interface component (widget). Every visual and behavioral characteristic of a widget is defined by an XPath expression, and is updated whenever the result of the XPath expression changes.

What user-interface toolkits are supported?

The reference implementation of xidgets was programmed with Java 1.5 with a binding for the Swing widget toolkit. Bindings for C#/WinForms and C++/Qt are in progress.

What is required to begin using xidgets?

Eclipse Java IDE, Java 1.5+ and Subversion 1.6.

Is the xidget framework extensible?

Yes. The xidget framework was designed to be extensible in three areas: widgets, scripting and data-model.

What are the terms of the software license?

Xidgets are distributed under the Apache 2.0 software license agreement which allows individuals and companies to develop and market proprietary software that uses the xidget libraries. However, any modifications to the xidget libraries are open-source.
