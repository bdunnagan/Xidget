MenuItem

Overview

The menuItem xidget creates an item in a menu. It behaves like a button xidget. The following example shows a menu containing two menu items and a separator:

<menu> <menuItem label="'Save'"> <onPress> <print>'Save'</print> </onPress> </menuItem> <separator/> <menuItem label="'Exit'"> <onPress> <destroyXidget>$window</destroyXidget> </onPress> </menuItem> </menu>

Events

Event Description
onPress Called when the widget is selected.
