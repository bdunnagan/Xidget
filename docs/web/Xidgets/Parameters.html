Common Parameters

Overview

The xidget schema includes some parameters that are common to most xidgets. The xidget reference lists the common parameters that are allowed for each xidget. Many of the common parameters specify xpath expressions. Each parameter supports a subset of the XPath 1.0 return types. String values may be specified with any of the four XPath 1.0 return types, the xpath will be evaluated as if it is enclosed in the string() function.

The following table lists the common parameters.

Node Description
label An xpath that determines the label to show to the left of the xidget.
image An xpath that determines the label to show to the left of the xidget.
source An xpath whose first node stores the text of the xidget.
enable A boolean xpath that determines when the xidget is enabled.
editable A boolean xpath that determines when the xidget is editable.
visible A boolean xpath that determines when the xidget is available.
foreground An xpath whose string value contains the hexadecimal rgb foreground color.
background An xpath whose string value contains the hexadecimal rgb background color.
font An xpath whose string value gives the name of the font family.
fontStyle An xpath whose string value gives the name of the font style.
Syntax: (plain|italic|bold)
fontSize An xpath whose numeric value gives the font size in points.
tooltip An xpath whose string value gives the tooltip text.
error An xpath that returns a non-empty string when there is a validation error.
size An xpath whose string value determines the size.
Syntax: width(\s*),(\s*)height(\s*)(char|pixel)
