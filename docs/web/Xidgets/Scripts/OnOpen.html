OnOpen

Overview

The onOpen script can be added to any xidget. It is called before the xidget is bound and becomes visible.
See The Xidget Lifecycle for more information. The script can execute any script element, but typically it initializes the data on which the xidget will operate.

In the following example, the onOpen script creates the content of a table, as well as the parent element that holds the selected rows of the table.

<form> <onOpen> <create assign="rows"> <template> <rows> <row>Row 1</row> <row>Row 2</row> <row>Row 3</row> </rows> </template> </create> <create assign="selection" name="'selection'"/> </onOpen> <layout> <layoutFillX/> <layoutFillY/> </layout> <table> <selection parent="$selection"/> <rows>$rows</rows> <column> <source>.</source> </column> </table> </form>
