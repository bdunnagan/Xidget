Text

Overview

The text xidget implements a text input with one or more lines. The size node can set the number of rows and columns of text by specifying the char.

Examples

The following example creates a labeled text field with a width of eighty characters and two rows.

<tex size="'80,2char'"t>
  <label>'Username'</label>
  <source>$login/username</source>
</text>

The following example makes shows a text field which is editable when the $authenticate variable evaluates true.

<text>
  <editable>$authenticate</editable>
  <label>'Username'</label>
  <source>$login/username</source>
</text>
