Slider

Overview

The slider xidget is a linear, numerical selection widget. It can be oriented horizontally or vertically. The minimum and maximum values can be specified along with the decimal-point precision.

The following example creates a slider which selects values between 0 and 10 with one decimal point precision:

<slider> <minimum>0</minimum> <maximum>10</maximum> <precision>1</precision> <source>$value</source> </slider>

Parameters

Node Description
minimum The minimum value obtained with the slider fully to the left or top.
maximum The maximum value obtained with the slider fully to the right or bottom.
precision The number of decimal points of precision in the value.
