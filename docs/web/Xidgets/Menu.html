Overview

The menu xidget will create either a menu within a menubar or a popup menu. In either case, the schema is the same, and specifies the menu text and image (popup menus do not have images). Menus also support boolean enable and visible expressions.

A menu element is also used to define a sub-menu.

Within a menu, the separator element creates a visual separation between menu items.

Example

The following example shows a popup menu defined on a table.

<table>
  <selection parent="$selection"/>
  <rows>*</rows>
  <cell>
    <source>name()</source>
  </cell>
  
  <menu>
    <menuItem label="'Open Item'">
      <onPress>
        <print>printf( 'Open item: %s\n', name( $selection/*))</print>  
      </onPress>
    </menuItem>
  </menu>
</table>
