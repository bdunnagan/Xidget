Overview

The menu xidget will create either a menu within a menubar or a popup menu. In either case, the schema is the same. The menu xidget supports images via the image element. Menus can be conditionally enabled using the enable element, and shown/hidden using the visible element. Menus contain menuItem children which define the menu options.

Scripts

OnOpen Called before the form widget becomes visible.
OnClose Called after the form widget becomes invisible.
Trigger Called when an xpath condition is satisfied after the form is visible.
