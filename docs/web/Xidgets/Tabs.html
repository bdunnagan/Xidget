Tabs

Overview

The tabs xidget is a container for tabbed forms. The title parameter of each form child defines the title of the tab. Form children are added to the tabs xidget by calling createXidget and specifying the tabs form as the parent. Similarly, forms children are removed by calling destroyXidget. If the removable parameter is set to "true", then each tab will have a close button next to it, which will remove the tab.
