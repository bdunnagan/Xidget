Common Xidget Behaviors

Overview

The xidget schema includes some elements that are common to most xidgets. For example, the enable xpath is a boolean expression that determines when a xidget is enabled. A disabled xidget will appear "grayed-out" and will not accept user input.

Enablement

As mentioned above, the enable element contains a boolean expression. When the boolean expression is true, the xidget is enabled and will accept user-input. The xidget will give visual indication of whether it is disabled, usually by becoming "grayed-out".

Visible

Like the enable element, the visible element contains a boolean expression that determines when the xidget is visible. When the expression is false, the element is not visible.

Background and Foreground Colors

The background and foreground elements contain string expressions which return an rgb color string. The color string consists of 6 hex characters, one byte per channel. For example, the string "ff0000" translates to a value of 255 for the red channel, and a value of zero for the green and blue channels.

Font, FontStyle and FontSize

The font used by a xidget is determined by the font, fontStyle and fontSize elements. The font and fontStyle expressions return a string, while the fontSize expression returns a number. The font string contains the name of the font family. The fontStyle string contains one of: plain, italic or bold. The fontSize value is measured in points.

Label

The label element defines the label that will appear to the left of a xidget. Labels do not apply to container xidgets. The label element contains an expression that returns a string, or a node.

Tooltip

The tooltip element defines the tooltip for a xidget. The tooltip element contains an expression that returns a string, or a node.

Error

The error element contains an xpath expression that returns a non-empty string whenever there is a problem with the data the user has entered into the xidget. The string that is return contains the text of the error that will be displayed to the user.
