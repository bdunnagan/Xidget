Window

Overview

The window xidget is a top-level window. It is the only container that supports the menubar and toolbar xidgets. The title of the window is defined in the title node.

  <window title="'My Window'" size="400,300">
    <package>org.xidget.xaction</package>
    <package>org.xidget.layout.xaction</package>

    <onOpen>
      <assign name="images">images</assign>
      <assign name="xml">xml</assign>
      <create assign="history" name="'history'"/>
    </onOpen>
    
    <menubar>
      <menu label="'File'">
        <menuItem label="'Quit'">
          <onPress>
            <destroyXidget>$window</destroyXidget>
          </onPress>
        </menuItem>
      </menu>
      
      <menu label="'Edit'">
        <menuItem label="'Undo'">
          <image>$images/undo.gif</image>
          <enable>$history/@index > 0</enable>
          <onPress>
            <undo history="$history"/>
          </onPress>
        </menuItem>
        <menuItem label="'Redo'">
          <image>$images/redo.gif</image>
          <enable>$history/@index < count( $history/*)</enable>
          <onPress>
            <redo history="$history"/>
          </onPress>
        </menuItem>
      </menu>
    </menubar>
    
    <insert>$xml/forms/RootForm.xml/*</insert>
  </window>

Parameters

Title The title of the top-level window.

Scripts

OnOpen Called before the form widget becomes visible.
OnClose Called after the form widget becomes invisible.
Trigger Called when an xpath condition is satisfied after the form is visible.
