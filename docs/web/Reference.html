Xidget Documentation

Element Description
Window A top-level window xidget.
Form A general purpose xidget container.
Tabs A tab container with one tab for each form child.
Label A text and/or image label xidget.
Button A push button, check button, toggle button or radio button.
Combobox A combo-box or choice list xidget.
Slider A numerical value xidget.
Text A single or multi-line text xidget.
Table A table xidget.
Tree A tree xidget.
Menu A top-level menu xidget used in menubars, toolbars and popup menus.
MenuItem An item in a menu.

 

Scripting Documentation

Element Description
add Add one or more elements, references, or foreign-key references to one or more parents.
assign Assign a value or node-set to a variable.
delete Remove one or more elements from their respective parents.
context Execute a script in a nested context.
create Create one or more elements and either parent them or assign them to a variable.
for Iterate over a node-set or sequence of numbers.
if/else Conditional execute scripts.
select Select one or more scripts to execute based on the string value of an xpath.
set Set the value of one or more nodes.
